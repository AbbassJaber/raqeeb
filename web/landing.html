How it works · it's a real agent

One instruction. Perceive → reason → act.

From a single instruction, Raqeeb chooses and sequences its own tools, narrating its reasoning at every step. Not a chatbot, not a fixed script — the model decides each move.

Deterministic where it counts

Area, distance and the setback intersection are computed in a metric projection (EPSG:32636) on same-season, cloud-masked composites. Math, not a screenshot.

AI where it helps

A vision model reads the before/after crop and names the change; a second, skeptical model double-checks it for cloud, tide or harvest. It doesn't cry wolf.

Always finishes

Every AI step has a deterministic fallback, so a run always completes — and cached real cases render even with no network.
